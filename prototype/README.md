# UNIFLOW - Frontend Prototype 🌊

Welcome to the **UNIFLOW** frontend prototype! This is a complete, fully-linked suite of static HTML mockups capturing the bold, neo-brutalist design of the UNIFLOW application. It represents an honest, minimal-jargon approach to daily habit tracking and AI accountability.

## 📁 What's Included

The prototype consists of 8 fully designed, interconnected UI screens:
1. `design.html` - The Landing Page and marketing website.
2. `auth.html` - The Login and Signup flow.
3. `dashboard.html` - The main user dashboard with current flow and an AI Coach widget.
4. `habits.html` - The Habits Management tab to create/edit tasks and "High Stakes" penalties.
5. `coach.html` - The AI Assistant "Aura" chat interface.
6. `analytics.html` - The data insights and consistency heatmaps.
7. `billing.html` - The upgrade / pricing comparing the Free vs. Pro tier limits.
8. `settings.html` - User profile and app preferences.

## 🚀 How to Run the Prototype

**You do NOT need Node.js, Python, npm, or any complex terminal commands to test this.** 

Since this is a lightweight frontend prototype built entirely with pure HTML and TailwindCSS, you can run the entire application directly from your file explorer!

### The Exact Steps to Run:
1. Open your computer's File Explorer (or Finder on Mac).
2. Navigate to this project folder.
3. **Double-click on `design.html`**
4. It will instantly open in your default web browser (Chrome, Safari, Edge, etc.).

From the `design.html` landing page, you can freely click the **"Start UNIFLOW Free"** or **"Get Started"** buttons to securely navigate between all the other HTML pages just like a real website! 

## 🛠 Tech Stack Used
- HTML5
- Tailwind CSS (via CDN for rapid prototyping)
- Lucide Icons (Web Components)
- Vanilla Javascript (minimal, for hover states and transitions)
